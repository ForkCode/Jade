### What is Jade?

- templating language
- clean, minimal html
- originally packaged as a node module
- available for Python, Ruby etc.


### Installing Jade:

    $ npm install -g jade


### Command line interface:

    $ jade --help


#### Convert a Jade file to HTML file

    $ jade index.jade
    $ jade -P index.jade
    $ jade -P -w index.jade
